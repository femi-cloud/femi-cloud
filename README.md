## 🚀 Featured Projects

### 🤖 [SRE-GPT](https://devpost.com/femi13sanya)
> Autonomous Site Reliability Engineering agent — Google Cloud × Dynatrace Hackathon

An AI-powered SRE agent that monitors, diagnoses, and auto-repairs production incidents in real time. Integrates **Gemini 2.5 Flash** for root cause analysis and post-mortems, **Dynatrace MCP** for observability, and a cascaded auto-repair + rollback system. Ships with a glassmorphism Ops Center dashboard and a React/Vite interface with PDF export.

`FastAPI` `Gemini 3.1 Flash Lite` `Dynatrace MCP` `React/Vite` `PostgreSQL` `Google Cloud`

### 🏥 [MedVoice Africa](https://devpost.com/femi13sanya)
> Offline-first multimodal medical assistant — Edge AI for rural healthcare

An offline-first, multimodal medical assistant powered by **Gemma 4**, designed to bridge the healthcare gap in rural areas via edge-AI. No internet required — intelligence at the edge, where it's needed most.

`Gemma 4` `Edge AI` `Multimodal` `Mobile`
